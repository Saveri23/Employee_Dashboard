# 🚀 Employee Dashboard – React TypeScript Assignment

## 📌 Project Overview

This is a ReactJS + TypeScript application built as part of an assessment.

The application includes:

- Login Authentication
- API Integration (POST Request)
- Employee List Display
- Employee Details Page
- Camera Capture Feature
- Photo Result Page
- Salary Bar Chart
- Employee City Map

---

## 🛠 Tech Stack

- React + TypeScript
- Vite
- React Router
- Axios
- Tailwind CSS
- Recharts (Bar Graph)
- Leaflet (Map)
- Camera API (navigator.mediaDevices)
